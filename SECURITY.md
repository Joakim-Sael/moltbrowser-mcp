# Security

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, use [GitHub's private vulnerability reporting](https://github.com/Joakim-Sael/moltbrowser-mcp/security/advisories/new) to submit a report confidentially.

When reporting, please include:

- Type of issue (e.g. credential exposure, remote code execution, etc.)
- File paths and line numbers related to the issue
- Steps to reproduce
- Potential impact

You should receive a response within 72 hours.

## Scope

Security issues in the underlying Playwright browser engine should be reported to [Microsoft](https://github.com/microsoft/playwright-mcp/security).
